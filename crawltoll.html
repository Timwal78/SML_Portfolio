// HOW THE TOLL WORKS

1. Bot requests a page → HTTP 402 + machine-readable x402 payment terms
2. Agent sends X-PAYMENT header (USDC on Base)
3. Payment verified + settled via facilitator → content unlocks instantly
4. Every fetch = USDC in your wallet. Built-in revenue ledger + npx crawltoll stats
