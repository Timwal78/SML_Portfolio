## MCP Server

**[`mcp-x402/`](./mcp-x402)** — the first MCP server that pays for itself. AI
agents pay for ScriptMasterLabs data autonomously via the [x402
protocol](https://github.com/coinbase/x402): 40+ pay-per-call tools covering
market intelligence, SEC filings, federal grants/contracts, FDA/OSHA/EPA
records, and more, settled in USDC on Base or RLUSD on the XRP Ledger. No API
keys, no subscriptions — see [`mcp-x402/README.md`](./mcp-x402/README.md) for
setup and the full tool list.
